사용법

1. 상단 필터로 지역, 가격, 평형 선택 후 검색
2. 행 클릭 시 해당 위치로 지도 이동
3. "링크 전체 열기"로 최대 10개 매물을 한 번에 새 탭에서 확인
4. 별표 클릭으로 관심 매물 저장 (브라우저에 저장됨)
5. 메모 칸에 타이핑하면 자동 저장
6. "매물 추가" 탭에서 PDF 업로드 → 자동 텍스트 추출 → 확인 후 추가
7. 또는 수동으로 입력도 가능
